Términos de Uso

Bienvenido a Helping Teachers. Al utilizar este sitio web y sus herramientas educativas, aceptas estos términos de uso.

Uso exclusivo para maestros

Helping Teachers está diseñado para maestros, educadores y personal escolar adulto. Los estudiantes no deben crear cuentas ni usar la plataforma directamente sin supervisión del maestro.

Acceso y funciones guardadas

Puedes usar las herramientas sin crear una cuenta. Si las funciones de inicio de sesión están disponibles, la cuenta será necesaria solo para funciones opcionales de guardado, como guardar preferencias, listas, horarios o configuraciones de herramientas.

Información de estudiantes

Cuando uses herramientas que pidan nombres o listas de estudiantes, no escribas nombres completos ni información sensible de estudiantes. Usa nombres de pila, apodos, iniciales o un pequeño identificador de aula.

Uso permitido

Puedes utilizar las herramientas para fines personales, educativos y de aula. Aceptas no utilizar el sitio de manera indebida, interferir con su funcionamiento, intentar acceder a sistemas no públicos o usar las herramientas de una manera que viole requisitos de tu escuela, distrito o leyes aplicables.

Disponibilidad

Estas herramientas se ofrecen "tal como están", sin garantías de ningún tipo. Aunque procuramos mantenerlas disponibles y funcionales, no podemos garantizar servicio ininterrumpido ni funcionamiento libre de errores.

Cambios

Helping Teachers se reserva el derecho de actualizar, modificar o descontinuar cualquier herramienta o función sin previo aviso. Al continuar utilizando este sitio web, aceptas estos términos y cualquier actualización futura.
