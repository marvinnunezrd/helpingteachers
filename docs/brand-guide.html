Brand Guide

Helping Teachers — visual identity, colors, typography, and tone.

Concept

Helping Teachers is a simple educational tools dashboard for Pre-K, Kindergarten, and elementary teachers before middle school.

The brand should feel like a modern educational playground: friendly, organized, colorful, trustworthy, and easy to use.

Tagline: Simple Tools for Smarter Classrooms

Tone

Visual Style

Modern Educational Playground. Inspiration:

  • Educational blocks
  • Crayons and pencils
  • Chalkboards
  • Flashcards
  • School playground
  • Clean EdTech interfaces

AI Image Prompt

Use this base prompt for any AI-generated icon or illustration:

Modern Educational Playground, clean educational dashboard, rounded vector style, bright classroom colors, friendly teacher tools, simple EdTech visual system, Pre-K through elementary school, white or transparent background.

Avoid:

  • Photorealistic style
  • Anime
  • Dark themes
  • Exaggerated gradients
  • Chaotic elements
  • Overly childish or toy-like appearance
