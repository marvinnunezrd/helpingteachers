# Helping Teachers — Brand Guide

## Dominio oficial

https://helpingteachers.io

## Concepto

Helping Teachers es un dashboard simple de herramientas educativas para maestros de Pre-K, Kindergarten y primaria antes de middle school.

La marca debe sentirse como un playground educativo moderno: amigable, organizada, colorida, confiable y fácil de usar.

## Frase principal

Simple Tools for Smarter Classrooms

## Reglas para imágenes IA

Toda imagen o icono debe partir de esta dirección:

Modern Educational Playground, clean educational dashboard, rounded vector style, bright classroom colors, friendly teacher tools, simple EdTech visual system, Pre-K through elementary school, white or transparent background.

Evitar:
- Realismo fotográfico
- Anime
- Estilo oscuro
- Gradientes exagerados
- Elementos caóticos
- Apariencia demasiado infantil o de juguetería
